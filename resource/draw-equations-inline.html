Some examples

Occasionally you may need to display an equation in a webpage. I prefer to use function names rather than classical notation; but tastes vary. This technology allows smart typesetting for that equation; and it can adapt to any screen size and it works for blind users. This is built as HTML is, from tags.
Note these Greek letters are perfectly normal Greek letters in UTF-8. This approach is very transparent to speech readers and in this context saying the letters name is normal (i.e. Delta, Epsilon etc).

Background/ legacy

A long time ago there was a tool called tex 1 2. This was first published in 1978 by a Maths doctorate for writing maths papers/ proofs/ equations via a computer. The doctor is famous for his other publications such as 3 and publishing an idea literate-programming. This is a product of its time, and by modern standards not an easy to use tool.
There is an earlier format called troff that is still used to make man pages. This is a simple typesetting language, outputting to text terminals that were normally used with computers in the 1960s/1970s. Troff is not optimised for displaying complex typography, and neither are mono-space text terminals ~ which are also limited to monochrome in the 1970s.
More recently, 'internet generation' people who aren't limited to 1970's tech, can use MathML 4 5 spec 6. Although the spec has been around for a long time, the support in browsers is lower than I would have expected 7 (at 90%; not available for the deceased MSIE, or minority mobile browsers). There is a W3C JS polyfill 8 and also a thing in NPM ~ which doesn't have much traction 9.

  • Benefits: MathML is entirely readable, a text format that can be in-line in a HTML file, and supports some CSS declarations.
  • Liabilities: MathML is more verbose than other options, and to repeat, has borderline viable browser support (90%).
Details on MathML
  • Look at your equation in an online tool TeXZilla or MathMLcentral as a viewer external to your stack, and for testing what you typed.
  • A handy online tool that will show errors in your XML 1. The W3C validator is claimed to do MathML 2, and it does have detailed errors.
  • Some examples/ explainers 3 4 5 .
  • Some template CSS 6 a W3C spec 7.
  • Some MathML specific editors EquaThEque, Mathtype demo (not specifically MathML, but these demos allow mouse or finger sketching for your Equation)
  • A handy list of MathML tag names 8, if you need them for some reason.
Other options for rendering complex typography
  • Hair splitting option: use MathML in an ANNOTATION XML element, rather than a complete XML sub-document.
  • A more widely supported version is a SVG containing the letter outlines of the equation (96% of users). Care will be needed designing the SVG.
  • Recommended : Install MathJAX docs NPM git and supply equations as Latex, or MathML. Latex is a less easy to use format, there are many questions on stackoverflow about rendering specific equations.
  • Use ASCIIMath as its easier to type, but still rendered via MathJAX. Examples 9 (warn: http link).
