Ambition

I looked a company website, and they are highly committed to using JS for animation. As a maquette, I thought I try to repeat what they had in CSS. I know precisely how to do this transform in tools that are not a browser. Also, after a few iterations I could build it in OpenGL (the end of 90s version OpenGL has less sophisticated curve surfaces, so it would take be a bit longer to get curves and blends). To make most 3d game engine tests, I ought to add some light sources, and have reflections. I think this is outside of capacity for CSS at present.
This maquette is also a trial run for the twisting infinity ‚Äúplease hold‚Äù I would like to achieve just using CSS, and maybe SVG.
